1. Uso de @keyframes em CSS

As regras de @keyframe podem ser usadas para criar animações de itens e objetos CSS. Neste código, foi usado para o slideshow desta página. Mais informações podem ser conseguidas neste link. O exemplo de uso está abaixo:

/* imagem de slideshow */
.slide {
    max-width: 100%;
    animation: fadeIn 5s infinite;
}

/* Define o efeito de fade-in */
@keyframes fadeIn {
    0%{
        opacity: 0;
    }
    20%{
        opacity: 1;
    }
    80%{
        opacity: 1;
    }
    100%{
        opacity: 0;
    }
}

2. Uso dos parâmetros list-style-type, position e overflow em CSS

O parâmetro list-style-type permite editar o ícone que aparecerá numa lista HTML. Se usada a palavra none, nada será usado para indicar os tópicos. Mais informações podem ser conseguidas neste link.

O parâmetro position permite que se indique onde um objeto ficará na página, obrigatóriamente. Foi usado para fixar os navbars na página, por exemplo. Mais informações podem ser conseguidas neste link.

O parâmetro overflow indica o que ocorrerá com um conteúdo da página caso ela exceda seu espaço originalmente destinado. Mais informações podem ser conseguidas neste link.

Um exemplo de implementação está abaixo:

ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
    width: 12%;
    background-color: #6c1427;
    height: 100%;
    position: fixed;
    overflow: auto;
}

3. Uso de gradientes em CSS

Permite criar gradientes de cores no fundo de objetos num geral. Neste site, foi utilizado nas linhas de divisão (o <hr>). Mais informações podem ser conseguidas neste link. O exemplo está logo abaixo:

hr {
    background-image:linear-gradient(to right, #6c1427, #d7284e, #6c1427);
}

4. Modificador nas cores de links

Na tag <a>, é possível alterar, via CSS, as cores padrão dos links. O parâmetro .link especifica a cor no geral, .visited quando a página já foi clicada e .active no momento que é clicada Mais informações podem ser conseguidas neste link. O exemplo de uso está a seguir:

.nos a:link {
    color: #e50a43;
}

.nos a:visited {
    color: #d7284e;
}

.nos a:active {
    color: #330a15;
}

5. Uso de sombras em elementos

Em CSS, é possível criar sombra em elementos usado o parâmetro box-shado. Mais informações podem ser conseguidas neste link. O exemplo de uso está a seguir:

.grupo {
    box-shadow: 1px 1px 10px 2px #b9b9b9;
}

7. Uso de funções Javascripto

Foi usada uma função em Javascript para fazer o carrossel de imagens encontrado na página principal. A implementação do código está mostrada abaixo.

var myIndex = 0;
carousel();

function carousel() {
    var i;
    var x = document.getElementsByClassName("slide");
    for (i = 0; i < x.length; i++) {
       x[i].style.display = "none";  
    }
    myIndex++;
    if (myIndex > x.length) {myIndex = 1}    
    x[myIndex-1].style.display = "block";  
    setTimeout(carousel, 5000);    
}
